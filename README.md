E-commerce store built with Next.js and Stripe checkout.

To run your site locally, use:

```
npm run dev
```

To run it in production mode, use:

```
npm run build # To test all pages
npm run start
npm run dev # To run locally
```

To run it in localhost, use:

```
http://localhost:3000

```
